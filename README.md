### Prerequisites

-   Node.js (v18+)
-   pnpm (recommended) or npm/yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/gabehamasaki/my-portfolio.git
    cd my-portfolio
    ```

2.  **Install dependencies**
    ```bash
    pnpm install
    ```

3.  **Run development server**
    ```bash
    pnpm dev
    ```

4.  **Build for production**
    ```bash
    pnpm build
    ```
